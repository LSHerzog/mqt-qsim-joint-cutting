# MQT qsim for Joint Cutting

An expansion to [qsim](https://github.com/quantumlib/qsim/tree/master) which allows for joint cutting of multiple gates during Hybrid Schr√∂dinger-Feynman Simulation (`qsimh`). For more general information on the original repository, refer to the [readme](https://github.com/quantumlib/qsim/blob/master/README.md). Regarding the changes for joint cutting, refer to the [documentation](https://github.com/LSHerzog/mqt-qsim-joint-cutting/blob/joint-cutting-laura/docs/joint_cutting.md).

If you have any questions, feel free to contact us via [quantum.cda@xcit.tum.de](mailto:quantum.cda@xcit.tum.de) or by
creating an issue on [GitHub](https://github.com/LSHerzog/mqt-qsim-joint-cutting/issues).
